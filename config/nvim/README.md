# Neovim example

This representative setup targets Neovim 0.12 and the current
`nvim-treesitter` API. It bootstraps lazy.nvim, pins the reviewed plugin set in
`lazy-lock.json`, and keeps the example intentionally smaller than the private
workstation configuration.

The snapshot demonstrates navigation, search, Git signs, statusline, comments,
automatic pairs, and parser-based highlighting. Language-server, debugger, and
project formatter ownership is described in
[`docs/tooling.md`](../../docs/tooling.md) rather than duplicated here.
